# MVP1 Scope

## Goal

Deliver a working prototype that demonstrates reusable issue capture, UAT Mode Lite, evidence, export, and optional Jira posting.

## In Scope

- central RAP service
- reusable embedded component
- central admin app
- issue header and context
- evidence metadata and content handling
- UAT case header and steps
- assignment
- execution header
- copy and paste boxes
- create issue from failed test
- AI export v1
- optional Jira issue or JSM request creation

## Out of Scope

- full campaign orchestration
- step-level analytics dashboarding
- complex approval flows
- Jira synchronization back to SAP
- enterprise-grade notification framework

## MVP1 Success Criteria

- user can launch the framework from a Fiori app
- user can report an issue with context
- user can attach evidence
- user can execute a UAT case
- user can mark the case failed and create a linked issue
- user can generate export output
- user can create a Jira record from SAP
- admin app shows issues and assigned cases
