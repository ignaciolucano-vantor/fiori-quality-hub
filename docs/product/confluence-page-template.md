# Confluence Page Template

## Title
Fiori Quality Hub | MVP1 Overview

## Summary
Fiori Quality Hub is a reusable SAP Fiori framework for issue reporting, UAT Mode Lite, evidence capture, AI-ready export, and optional Jira ticket creation.

## Why This Exists
- reduce manual issue handoff
- improve traceability
- support reusable integration across Fiori apps
- accelerate triage with structured context

## MVP1 Scope
- Issue reporting
- UAT Mode Lite
- Evidence
- AI export v1
- Optional Jira creation

## Repo and Planning Links
- GitHub repository
- GitHub Project
- Jira board or project
- Demo assets

## Architecture Summary
- central RAP backend
- reusable embedded frontend library
- central admin app
- consuming app adapter

## Demo Flow
1. Launch from app
2. Execute UAT case
3. Fail case
4. Capture evidence
5. Create issue
6. Generate export
7. Create Jira ticket
8. Review in admin app
