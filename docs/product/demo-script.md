# Demo Script

## Demo Goal

Show a full defect capture and handoff flow inside SAP Fiori with optional Jira creation.

## Demo Steps

1. Open the consuming Fiori app.
2. Launch Fiori Quality Hub from the embedded entry point.
3. Open an assigned UAT case.
4. Copy test values from the copy boxes.
5. Execute the scenario and mark it Failed.
6. Attach evidence and comments.
7. Create a linked Issue.
8. Generate AI-ready export text.
9. Create a Jira record automatically.
10. Open the admin app and show the linked records and Jira trace.

## Demo Talking Points

- Reusable across multiple Fiori apps
- Better traceability
- Faster handoff from testing to engineering
- Structured context instead of manual ticket writing
- Agent-friendly export for faster remediation
